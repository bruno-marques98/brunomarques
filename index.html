Personal Finance Management Web Application

Designed and developed a full-stack personal finance management web application that enables users to securely manage income and expenses, categorize transactions, and analyze financial data through a clean and responsive interface.

Core Functionalities

  • User authentication using JWT (secure token-based access).
  • CRUD operations for financial transactions (income and expenses).
  • Transaction categorization with dynamic category management.
  • Real-time filtering and searching of transactions by text and type.
  • Integration with RESTful APIs for all data operations.
  • Automatic authorization header injection via HTTP interceptors.
  • Centralized error handling and API response validation.
  • Modular and scalable architecture designed for future expansion.

Frontend (Angular)

  • Angular (standalone components architecture).
  • TypeScript.
  • Angular Forms (template-driven and reactive).
  • HTTP Client for REST API consumption.
  • HTTP Interceptors for JWT token handling.
  • Environment-based configuration for API URLs.
  • Clean, financial-app-oriented UI with a neutral grey design.
  • Component-based filtering and state management.

Backend (.NET)

  • ASP.NET Core Web API.
  • C#.
  • RESTful API design.
  • JWT authentication and authorization.
  • Dependency Injection.
  • Clear separation of concerns (Controllers, Services, Models).
  • SQL-based persistence using Entity Framework and SQL Server.

Security

  • JWT-based authentication.
  • Authorization headers automatically attached to API requests.
  • Environment-specific configuration for sensitive values.

Architecture & Best Practices

  • Full separation between frontend and backend.
  • Strongly typed models.
  • Reusable services for API communication.
  • Interceptor-based handling of cross-cutting concerns.
  • Scalable, feature-based project structure.
